Clickbait Blocker

General Options

Set relevance (confirmed clickbaits minus infirmed clickbaits):

* This value is taken into consideration when an article is considered clickbait or not. E.g: if you set a relevance value of 10, then all articles with the number of "confirmed as clickbait" votes higher than 10 will be hidden. All articles with the number of "confirmed as clickbait" votes lower than 10 will be shown.

Whitelist domains (0)

* A whitelist domain is a domain you trust. We'll never hide articles on this domain no matter the relevance.

Currently you don't have any domains whitelisted

Add custom relevance for certain domains

* For a domain in this list the custom relevance you set will override the general relevance. E.g: if you set a custom relevance of 20 for www.example.com and your general relevance is 10, then all articles on www.example.com, with the number of "confirmed as clickbait" votes higher than 20 will be hidden. All articles with the number of "confirmed as clickbait" votes lower than 20 will be shown.
